# 🛒 ONLINESHOP 

### 🤔 O que é ONLINESHOP ?

<p>OnlineShop é um Ecommerce com modelo B2B2C, no qual lojas terceirizadas, que estão cadastradas, vendem produtos para os consumidores finais.</p>
